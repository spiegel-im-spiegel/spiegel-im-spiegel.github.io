Request for Comments

  • RFC 1847 - Security Multiparts for MIME: Multipart/Signed and Multipart/Encrypted
  • RFC 3156 - MIME Security with OpenPGP (update RFC 2015)

  • RFC 4880 - OpenPGP Message Format (obsoletes RFC 1991 and RFC 2440)

    • RFC 5581 - The Camellia Cipher in OpenPGP (update RFC 4880)
    • RFC 6637 - Elliptic Curve Cryptography (ECC) in OpenPGP (update RFC 4880)

  • draft-ietf-openpgp-rfc4880bis - OpenPGP Message Format

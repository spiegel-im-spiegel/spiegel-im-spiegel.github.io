gpgpdump 0.3.6 をリリースした

Tags: #golang #gpgpdump #openpgp #tools
 :        
History in GitHub Page

OpenPGP パケットの内容を視覚化する gpgpdump の 0.3.6 をリリースした。

  • Release v0.3.6 · spiegel-im-spiegel/gpgpdump

実は本家 pgpdump のほうに Issue が上がっていて，件のパケットをうちでも試してみたら出力がダメダメだったのだ。 なので今回はバグ修正版。

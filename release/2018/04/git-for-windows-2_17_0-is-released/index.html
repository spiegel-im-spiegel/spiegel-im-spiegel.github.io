Git for Windows 2.17.0 がリリース（OpenSSL セキュリティアップデートを含む）

Tags: #git #openssl #security #tools #vulnerability #windows
 :     
History in GitHub Page

Git for Windows 2.17.0 がリリースされた。

  • Release Git for Windows 2.17.0 · git-for-windows/git

OpenSSL v1.0.2o はセキュリティ・アップデートを含んでいる（深刻度：CVSSv3 Base 評価 で 5.9 【警告】）。

  • JVNVU#93502675: OpenSSL に複数の脆弱性
  • JVNDB-2017-011252 - JVN iPedia - 脆弱性対策情報データベース

HTTPS アクセスに OpenSSL を選択している人は早めのアップデートをお勧めする。

アップデートは計画的に。

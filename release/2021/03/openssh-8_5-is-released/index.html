OpenSSH 8.5 のリリース【セキュリティ・アップデート】

Tags: #openssh #security #vulnerability
 :        
History in GitHub Page

先週の話で恐縮だが， 2021-03-03 にリリースされた OpenSSH 8.5 で CVE-2021-28041 の脆弱性の改修がされていたらしい。

ssh-agent のソケット周りとかヤバいぢゃん。

CVSSv3 評価は以下の通り。

  • CVSS:3.1/AV:N/AC:L/PR:N/UI:N/S:U/C:H/I:H/A:H
  • 深刻度: 緊急 (Score: 9.8)
基本評価基準 評価値
攻撃元区分 ネットワーク
攻撃条件の複雑さ 低
必要な特権レベル 不要
ユーザ関与レベル 不要
スコープ 変更なし
機密性への影響 高
完全性への影響 高
可用性への影響 高

実は Ubuntu で，これを修正したらしい OpenSSH 8.3 バックポート・パッチが出てたので気付いた次第。

  • USN-4762-1: OpenSSH vulnerability | Ubuntu security notices | Ubuntu

まぁ，アップデートは計画的に，ということで。

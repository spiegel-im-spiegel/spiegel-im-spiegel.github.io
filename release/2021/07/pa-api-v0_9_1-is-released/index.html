PA-APIv5 クライアント・パッケージ v0.9.1 をリリースした

Tags: #amazon #golang #module #pa-api #package
 :        
History in GitHub Page

ちょっとした改修の pull request をいただいたので，これを反映した Amazon Product Advertising API v5 (PA-APIv5) クライアント・パッケージ spiegel-im-spiegel/pa-api の v0.9.1 をリリースした。 と思ったのだが，何故か v0.9.0 のアナウンスをしてなかったみたいなので，併せて紹介する。

  • Release v0.9.0 · spiegel-im-spiegel/pa-api · GitHub
  • Release v0.9.1 · spiegel-im-spiegel/pa-api · GitHub

v0.8.0 をリリースしたときに「いっそ interface 型で汎化してしまうか」と書いたのだが，実際に v0.9.0 では Marketplace 型を

type Marketplace interface {
    String() string   // marketplace name (www.amazon.com)
    HostName() string // hostname (webservices.amazon.com)
    Region() string   // region (us-east-1)
    Language() string // language code (en_US)
}

と interface 型で定義しなおすことで自前でマーケットプレイス情報を定義できるようにした。 なお，元々の列挙型は MarketplaceEnum で再定義している。

私が公開しているパッケージでは珍しく pull request をいただくことが多いのだが，使っていただけるのなら幸いである。

gpgpdump v0.6.5 をリリースした

Tags: #gpgpdump #openpgp #tools
 :        
History in GitHub Page

OpenPGP パケットの内容を可視化する gpgpdump の v0.6.5 をリリースした。

  • Release v0.6.5 · spiegel-im-spiegel/gpgpdump · GitHub

ドラフト版 RFC 4880bis の仕様に追随できてない部分があったので，署名パケット（Tag 2）内のサブパケットの解析処理をいくつか修正した。

そろそろちゃんとテストを書かないとなぁ。 一応 RFC 4880bis が正式版になるまでには何とかしようという気はある。

まぁ鼻の先は動いているので気長にお付き合いください。

  • GitHub に登録されている OpenPGP 公開鍵の情報を取得する

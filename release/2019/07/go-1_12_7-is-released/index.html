Go 1.12.7 がリリースされた

Tags: #golang #language #programming #ubuntu
 :  
History in GitHub Page

Go 1.12.7 がリリースされた。 セキュリティ・アップデートはなし。

  • Go 1.12.7 and Go 1.11.12 are released - Google Group

さらっと書いてるがマイルストーンを見ると結構ヤバげな内容なんだよねぇ。

例によって Ubuntu の APT が提供する Go コンパイラは2世代も古くて使いものにならないため1，ダウンロードページから go1.12.7.linux-amd64.tar.gz を取ってきて任意の場所に展開する。

$ cd /usr/local/src
$ sudo curl "https://dl.google.com/go/go1.12.7.linux-amd64.tar.gz" -O
$ cd ..
$ sudo unlink go # 以前の Go が入っている場合
$ sudo tar xvf src/go1.12.7.linux-amd64.tar.gz
$ sudo mv go go1.12.7
$ sudo ln -s go1.12.7 go
$ go version # /usr/local/go/bin にパスが通っている場合
go version go1.12.7 linux/amd64

ほい。 ひと仕事終わり。

アップデートは計画的に。

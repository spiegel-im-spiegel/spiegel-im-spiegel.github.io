例によって…

Ubuntu の APT で管理している Go コンパイラは古いので，ダウンロードページからバイナリ（go1.19.3.linux-amd64.tar.gz）を取ってきてインストールすることを推奨する。 以下は完全手動での作業例。

$ cd /usr/local/src
$ sudo curl -L "https://go.dev/dl/go1.19.3.linux-amd64.tar.gz" -O
$ cd ..
$ sudo unlink go # 以前の Go が入っている場合
$ sudo tar xvf src/go1.19.3.linux-amd64.tar.gz
$ sudo mv go go1.19.3
$ sudo ln -s go1.19.3 go
$ go version # /usr/local/go/bin にパスが通っている場合
go version go1.19.3 linux/amd64

Windows は Scoop 経由で OK

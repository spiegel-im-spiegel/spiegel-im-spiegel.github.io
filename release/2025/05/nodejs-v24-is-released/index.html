Node.js v24 がリリースされた

Tags: #nodejs #programming #react
 :        
History in GitHub Page

最近仕事で React なんか弄ってるもんで Node.js の動向も押さえておこうかな，と。

  • Node.js — Node v24.0.0 (Current)

この記事について Kagi Assistant に要約してもらった。

というわけで Node.js v24 は 2025-10 以降に LTS になる予定である。 Node.js の LTS は通常30ヶ月間サポートされる。 現在 LTS として運用されているバージョンは v22 と v20 である（v20 は 2026-04 まで）。 また v18 は 2025-04 に EoL (End-of-Life) になっている。

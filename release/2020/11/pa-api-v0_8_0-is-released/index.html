PA-APIv5 クライアント・パッケージ v0.8.0 をリリースした

Tags: #amazon #golang #module #pa-api #package
 :      
History in GitHub Page

ありがたいことに pull request をいただきまして， Amazon Product Advertising API v5 へアクセスできる Go 言語用パッケージ spiegel-im-spiegel/pa-api の v0.8.0 をリリースした。

  • Release v0.8.0 · spiegel-im-spiegel/pa-api · GitHub

今回は Marketplace 定義の追加のみ。

spiegel-im-spiegel/pa-api では Marketplace を列挙型で定義しているのだが，頻繁に追加や変更があるなら上手いやり方ではないかもしれない。 いっそ interface 型で汎化してしまうか。 手を入れるのは簡単だが，今のところは考え中。

History in GitHub Page

Git v2.26.2 を含む複数のバージョンがリリースされた。

  • [Announce] Git v2.26.2 and others

対象となるのは 2.17.x から 2.26.x までの各マイナーバージョン。 前回の CVE-2020-5260 が直りきってなかったってことでいいのかな？

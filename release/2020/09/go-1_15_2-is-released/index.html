Go 1.15.2 がリリースされた

Tags: #golang #language #programming #ubuntu
 :        
History in GitHub Page

Go 1.15.2 がリリースされた。

  • Go 1.15.2 and Go 1.14.9 are released

セキュリティ・アップデートはなし。

例によって Ubuntu の APT で管理している Go コンパイラは古いので，ダウンロードページからバイナリ（go1.15.2.linux-amd64.tar.gz）を取ってきて手動でインストールすることを強く推奨する。 以下は手動での作業例。

$ cd /usr/local/src
$ sudo curl -L "https://golang.org/dl/go1.15.2.linux-amd64.tar.gz" -O
$ cd ..
$ sudo unlink go # 以前の Go が入っている場合
$ sudo tar xvf src/go1.15.2.linux-amd64.tar.gz
$ sudo mv go go1.15.2
$ sudo ln -s go1.15.2 go
$ go version # /usr/local/go/bin にパスが通っている場合
go version go1.15.2 linux/amd64

アップデートは計画的に。

「リリース情報」セクションを開設した

Tags: #section #site
 :        
History in GitHub Page

ここのところ GitHub リポジトリ で公開するパッケージやツールが増えてきたので，リリース情報を告知するための「リリース情報」セクションを開設することにした。 ドゾ宜しく。

つっても自作のものは殆どゴミ屑みたいなものばっかりなので，他にも「しっぽのさきっちょ」で挙げていた脆弱性情報なんかも今後は「リリース情報」セクションで扱うことにする（ただし過去記事の移動はしない）。

他にも気になるツールやサービスなんかも紹介するかもしれない。 その辺は気分次第で。

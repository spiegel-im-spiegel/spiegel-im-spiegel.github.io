徐々に再起動中

History in GitHub Page

最終更新日から2ヶ月ぶりのご無沙汰です。

個人的に色々ありまして，恥ずかしながら帰ってまいりました。 紅塵の巷にて徐々に再起動中です。 溜まってたネタはこれからボチボチ消化していく予定。

ゲームは4ヶ月くらい放置してたら血盟から外れてました（当たり前）。 再申請してこちらもゆっくりリハビリしてます。

今後の活躍にご期待下さい（笑）

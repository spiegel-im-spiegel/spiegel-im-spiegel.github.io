夜のセットリスト

Tags: #music
 :        
History in GitHub Page

昨年に「仕事中の BGM」「2024年のクリスマスソング・リスト」という記事を書いたが，ふと「夜」をテーマにしたセットリストを作ってみようと思い立った。 とりあえず，スマホに入ってる楽曲から選んでみた。

なお，順番に深い意味はない。 どうせシャッフルしてエンドレス再生するし（笑）

  1. Stellar Stellar（by 星街すいせい）
    → 6th fes. STAGE1 のアレを見ちゃったら楽曲を買わんわけにいかんじゃろ
  2. ムーンライト（by 星街すいせい）
    → 実は ReGLOSS のカバー MV を見て好きになった曲。いつか ReGLOSS 版も出ないかねぇ
  3. HELP!!（by こぼ・かなえる）
    → こっちはご本人の MV を見て衝動買いした。夜か？ と言われると微妙だが，悶々とする感じがなんとなく
  4. 夜の歌（「カードキャプターさくら」挿入歌, by 大道寺知世）
    → 実は私は知世派っす
  5. 花になって - Be a flower（「薬屋のひとりごと」ED曲, by 緑黄色社会）
    → 色事と政治は夜に行われる。なんちゃって
  6. Get Wild（「シティーハンター」ED曲, by TM Network）
    → 夜の歌というか帰宅曲？ Get Wild 退勤（笑）
  7. 歩いて帰ろう（by 斉藤和義）
    → これも帰宅曲だな。徒歩で帰るときによく聴いている
  8. ノーザンクロス（「マクロスF」挿入歌, by シェリル・ノーム）
    → シェリル・ノーム（May’n）さんの曲の中ではかなり好きな曲。これは全く余談だが，はくちょう座を「北十字星」と呼んだりする
  9. ファタール（「推しの子」OP曲, by GEMN）
    → これも何となくのイメージ。この曲については「音乃瀬奏」のショート動画がお気に入りだったり。フルバージョンが出ないかなぁ
  10. ひとりぼっちが好き（by 飯島真理）
    → これは本当に夜に聴くと切なくなるかもしれない
  11. よろこべ友よ恐れなく（by イルカ）
    → 夜というかクリスマスソングかな。原曲はイギリスだったかの賛美歌（“God Rest Ye Merry Gentlemen”）らしい
  12. 月光（by 鬼束ちひろ）
    → 昔，カラオケで歌おうとしたがサビしか分からなかった悲しい過去がある
  13. 宙船（by 中島みゆき）
    → やっぱ「宙船」は中島みゆきさんじゃないと
  14. 明日の私に幸あれ（「ギルドの受付嬢ですが、残業は嫌なのでボスをソロ討伐しようと思います」ED曲, by ナナヲアカリ）
    → アニメは見てない。明日残業しないために残業する。泣いてまうやろ（笑）

全14曲，54分半ほど。 いかがだったであろうか。 しばらくこれを聴くかな。

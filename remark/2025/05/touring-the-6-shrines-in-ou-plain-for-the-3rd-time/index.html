意宇六社巡り3回目（八雲立つ風土記の丘 遺跡めぐり 7）

Tags: #bike #matsue #osanpo-camera #photography #ruins #shrine
 :        
History in GitHub Page

みなさん GW いかがお過ごしでしょうか。 私は風邪を引いてしまいまして。 どうにか体調も戻ってきたので，そろそろ自転車で遊びたいかなっと。

今朝の最低気温が11℃，日中の最高気温が20℃。 西寄りの風が 7〜8m/s とちょっと強めだけど，まぁ大丈夫だろう。

今回は約1年ぶりに意宇六社1 を回ろうかな，と。

田和山遺跡 → 眞名井神社

今回は起点を田和山遺跡とした2。

ここからバイパス沿いに東に向かうと乃木二子塚古墳がある。

さらにバイパス沿いを東進し，松江工業高校から南下すると山代二子塚古墳がある。

古墳の敷地内は自転車進入禁止なので遠目から写真を撮ってみた。

山代二子塚古墳は出雲地方最大級の前方後方墳で，出雲の東半分の首長の墳墓と推測される。 また，おそらくは出雲国造（またはその血縁者）ではないか，とも考えられている。 この辺は「「東西出雲の王」を聴講する」や「「出雲国造の成立と出雲臣」を聴講する」あたりを参考にどうぞ。

ついでに大庭鶏塚古墳にも寄ってみた。

本当は山代方墳の写真も撮りたかったが，場所がよく分からず断念した。 山代の古墳群は改めて見に行くことにしよう。

さて，眞名井神社に到着。

鳥居の先の石段を登ると私のなけなしの体力が削られるので，今回はパス。 このまま揖夜神社へ向かおう。

眞名井神社 → 揖夜神社

特に寄り道もなく揖夜神社に到着。

おっ，工事が終わってる。 こちらも写真を撮るだけ。 参拝はまたいずれということで。

揖夜神社 → 六所神社

休憩なしでこのまま六所神社へ向かう。

おっ。 特急やくもだ。

この新型やくもは随分前に導入されたんよね。 そうそう，これこれ。

意宇川の鳥など眺めながら移動。

六所神社に到着。

ここも写真を撮るだけ。 次行こう，次。

六所神社 → 熊野大社（八雲温泉）

ここからはいつもの道行き。 日吉切通しを経由して

熊野大社に到着。

この時点で正午前だったので，先に昼飯食って温泉に入るか。 昼飯は久しぶりの安分亭。

猪ウマウマ。 お腹が落ち着いたところで八雲温泉へ。

木次乳業さん，いつもお世話になっています。

さっぱりしたところで熊野大社へお参りしておこう。

お稲荷さんにもお参りしておこう。

お稲荷さま。 お金持ちでなくてもいいけど，お金で困らない生活がしたいです（笑）

熊野大社 → 神魂神社 → 八重垣神社

次は神魂神社へ。

今年も意宇川で鯉のぼりを見かけた。

なんか河原でイベントやってるな。 楽しそうな親子連れを眺めながら北上する。

神魂神社に到着。

そういえばこの参道の桜を今年は見なかったな。 来年こそは！

本当はこちらもお参りしたかったのだが，熊野大社で小銭が尽きてしまった。 こちらもまた今度。

というわけで，このまま八重垣神社へ向かう。

これにて意宇六社巡り完遂である。

明るいうちに帰ろう

意宇六社巡りを完遂したので，とりあえずコメダ喫茶店まで移動して糖分補給。

ここで気がついたのだが，自転車用のライトを忘れたようだ。 というわけで，明るいうちに帰ることにした。 宍道湖を眺めながら帰路につく。

おー。 出雲空港に向かう飛行機の飛行機雲かな。

今日も楽しく遊びました。 明日は月曜日だけど，まだまだ連休は続く。

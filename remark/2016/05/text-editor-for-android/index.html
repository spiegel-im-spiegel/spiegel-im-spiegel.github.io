どうも Jota+ ってのがいいらしい。

Jota+ 自身は無料だが PRO-KEY を購入して activation すると色々いいことがある。 まず広告表示を無効にできる。 あとストレージのアクセスに Storage Access Framework が使えるようになる。 これは特にクラウド・ストレージを利用する際に便利だ。

Jota+ 自身も内臓機能でクラウド・ストレージにアクセスできるのだが，サービスごとに connector アプリを入れる必要がある。 SAF を有効にしておけばバックグラウンドで Box や Dropbox を起動した上で SAF 経由で各サービスにアクセスできる。

って，これって少なくともテキスト情報に限れば Evernote や Google Keep は要らないってことか。 善き哉。

そうそう。 Jota+ は既定で Google Analysis による情報収集を行っている。 これは設定で無効にできるので，気に入らないのであれば無効にすることをお薦めする。

新しい OpenPGP 鍵サーバが Launch したらしい

Tags: #cryptography #management #openpgp #pki
 : 
History

OpenPGP のメーリングリストより。

  • Launching a new keyserver on keys.openpgp.org!
  • Launching a new keyserver! - keys.openpgp.org
  • sequoia-pgp / hagrid · GitLab

ほほう。 Rust ベースなのか。 面白いな。

現行の OpenPGP 鍵サーバ同士は peer-to-peer で同期しているが，新しい keys.openpgp.org はこれらとは別のネットワークを形成するようだ。

さらに

ということで Enigmail や OpenKeychain と連携するのであれば期待できそうな感じである。 ただし，いまのところ GnuPG との相性がイマイチのようなので，もう少し様子を見たいところではある。

これで思い出すのが，かつての OpenPKSD だが，日本主導でフル Ruby で組まれていて割と期待してたんだが，世界的にはあまり注目されないまま閉鎖されたんだよなぁ。 今回の keys.openpgp.org がそうならないことを祈るばかりである。

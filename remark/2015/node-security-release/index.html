Tags: #nodejs #security #vulnerability
 :        
History in GitHub Page

前々から予告があったが node.js に関する複数の脆弱性に対するアップデートが行われた。

  • December Security Release Summary | Node.js
  • Node v0.10.41 (Maintenance) | Node.js
  • Node v0.12.9 (LTS) | Node.js
  • Node v4.2.3 “Argon” (LTS) | Node.js
  • Node v5.1.1 (Stable) | Node.js

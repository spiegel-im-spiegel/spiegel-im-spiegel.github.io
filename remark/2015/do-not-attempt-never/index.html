History in GitHub Page

Hugo гҒ§ HTML ж§Ӣж–Үж··гҒҳгӮҠгҒ® Markdown гӮ’дёҠжүӢгҒҸеҮҰзҗҶгҒ§гҒҚгӮӢгҒӢе®ҹйЁ“гҖӮ

з§ҒгҒҜе—ңеҘҪгҒҢд»–гҒ®дәәгҒЁгҒҜе°‘гҒ—гӮәгғ¬гҒҰгӮӢгҒҝгҒҹгҒ„гҒ§пјҲгғһгӮӨгғҠгғјеҝ—еҗ‘гҒЁгҒ„гҒҶгӮҸгҒ‘гҒ§гҒҜгҒӘгҒ„гӮҲпјүд»–дәәж§ҳгҒҢи–ҰгӮҒгӮӢгӮӮгҒ®гӮ’йқўзҷҪгҒ„гҒЁж„ҹгҒҳгӮӢгҒ“гҒЁгҒҢе°‘гҒӘгҒ„гҒ®гҒ гҒҢпјҢгҒ“гӮҢгҒҜгҒҫгҒҒгҒҫгҒҒйқўзҷҪгҒӢгҒЈгҒҹгҖӮ

гӮҝгӮӨгғҲгғ«гҒӢгӮүгҖҺеӯҗгҒ©гӮӮгҒҢдҪ“йЁ“гҒҷгӮӢгҒ№гҒҚ50гҒ®еҚұйҷәгҒӘгҒ“гҒЁгҖҸгҒҝгҒҹгҒ„гҒӘгҒ®гӮ’йҖЈжғігҒ—гҒҹгҒ®гҒ гҒ‘гҒ©пјҢгҒқгҒҶгҒ§гҒҜгҒӘгҒҸгҒҰпјҢз§‘еӯҰеҸІгҒ® trivial гҒӘи©ұйЎҢгӮ’йқўзҷҪгҒҠгҒӢгҒ—гҒҸзҙ№д»ӢгҒҷгӮӢжј«з”»гӮүгҒ—гҒ„гҖӮ гҒ“гҒҶгҒ„гҒҶгҒ®йқўзҷҪгҒ„гҒ‘гҒ©йЈҪгҒҚгӮӢгҒ®гӮӮж—©гҒ„гӮ“гҒ гӮҲгҒӘгҒҒгҖӮ

гҒӮгҒҒгҒ§гӮӮи’ёж°—ж»…иҸҢиЈ…зҪ®гҒ§и’ёгҒӢгҒ—иҠӢгӮ’дҪңгӮӢгҒ®гҒҜжҖқгҒ„гҒӨгҒӢгҒӘгҒӢгҒЈгҒҹгҒӘгҒҒгҖӮ гғӣгғігғһпјҢгӮўгӮ«гғҮгғҹгғғгӮҜгғ»гӮЁгғӘгӮўгҒ«еұ…гӮӢдәәгҒҜзөҢжёҲиҰіеҝөгҒҢгӮҒгҒЎгӮғгҒҸгҒЎгӮғгҒ гӮҲгҒӯпјҲиӨ’гӮҒиЁҖи‘үпјүгҖӮ

з§‘еӯҰгҒЁгҒҜеӨұж•—гҒ®жӯҙеҸІгҒ§гҒӮгӮҠй–“йҒ•гҒ„гҒ®жӯҙеҸІгҒ§гӮӮгҒӮгӮӢгҖӮ гҒІгҒЁгҒӨгҒ®жҲҗеҠҹгҒ®и¶іе…ғгҒ«гҒҜе…«зҷҫдёҮгҒ®иӣҚзҒ«гҒ®еҰӮгҒҚеӨұж•—гӮ„й–“йҒ•гҒ„гҒҢжЁӘгҒҹгӮҸгҒЈгҒҰгҒ„гӮӢгҖӮ гҒқгҒ®жҲҗеҠҹгӮ’жӯЈгҒ—гҒҸзҗҶи§ЈгҒҷгӮӢгҒҹгӮҒгҒ«гҒҜпјҢгҖҢйҒҺзЁӢгҖҚгҒЁгҒ—гҒҰгҒ®еӨұж•—гӮ„й–“йҒ•гҒ„гӮӮдҪөгҒӣгҒҰиӯҳгӮӢеҝ…иҰҒгҒҢгҒӮгӮӢгҖӮ гҒ“гӮҢгҒҢгҖҢз§‘еӯҰеҸІгҖҚгҒ®еӯҰе•ҸгҒЁгҒ—гҒҰгҒ®еӯҳеңЁж„Ҹзҫ©гҒ гҒЁжҖқгҒҶгҖӮ

гҒ“гӮҢгҒҜйҖҡеёёгҒ®гҖҢжӯҙеҸІгҖҚгҒЁгҒҜе°‘гҒ—ж„Ҹе‘іеҗҲгҒ„гҒҢйҒ•гҒҶгҖӮ йҖҡеёёгҒ®гҖҢжӯҙеҸІгҖҚгҒҜгҖҢеҗҢгҒҳиҪҚгӮ’иёҸгҒҫгҒӘгҒ„гҖҚгҒҹгӮҒгҒ®ж•ҷиЁ“гҒЁгҒ—гҒҰеҪ№гҒ«з«ӢгҒЈгҒҰгҒ„гӮӢгҒ®гҒ гӮҚгҒҶгҒҢпјҢз§‘еӯҰгӮ„жҠҖиЎ“гҒ«гҒҠгҒ„гҒҰгҒҜжҲҗеҠҹгҒЁеӨұж•—гӮ’иҫјгҒҝгҒ«гҒ—гҒҰи©•дҫЎгҒҷгӮӢеҝ…иҰҒгҒҢгҒӮгӮӢгҖӮ гҒӨгҒҫгӮҠгҖҢв—Ҝв—ҜгҒ§еӨұж•—гҒ—гҒҹгҖҚгҒЁгҒ„гҒҶгҒ®гҒӘгӮүпјҢгҒқгҒ®еӨұж•—гӮӮиҫјгҒҝгҒ§еҶҚзҸҫгҒ§гҒҚгҒӘгҒ‘гӮҢгҒ°гҒӘгӮүгҒӘгҒ„гҒ—пјҢгҒқгҒ®еӨұж•—гӮӮжҲҗеҠҹгҒЁеҗҢгҒҳгӮҲгҒҶгҒ«иЁҳиҝ°гҒ—гҒӘгҒҸгҒҰгҒҜгҒӘгӮүгҒӘгҒ„гҖӮ и«–ж–ҮпјқжҘӯзёҫ гҒЁиҖғгҒҲгҒҰгҒ„гӮӢдәәгҒ«гҒҜиЁұе®№гҒ§гҒҚгҒӘгҒ„гҒ гӮҚгҒҶгҒ‘гҒ©гҖӮ

гҒқгҒ—гҒҰгҒқгҒ®еӨұж•—гӮ’и©ҰгҒ—гҒҰгҒҝгӮҲгҒҶгҒЁиҖғгҒҲгӮӢгҒ®гҒҢгҖҢз§‘еӯҰиҖ…гҖҚгӮ„гҖҢжҠҖиЎ“иҖ…гҖҚгҒЁгҒ—гҒҰгҒ®жӯЈгҒ—гҒ„ж…ӢеәҰгҒ гҒЁжҖқгҒҶпјҲе®ҹйҡӣгҒ«гҒҜеӨ§жҳ”гҒ®е®ҹйЁ“гҒ®гҒӘгҒӢгҒ«гҒҜеҚұгҒӘгҒӢгҒЈгҒҹгӮҠпјҲзҸҫеңЁгҒ®пјүеҖ«зҗҶзҡ„гҒ«з„ЎзҗҶгҒӘгӮӮгҒ®гӮӮгҒӮгӮӢгҒ‘гҒ©гҒӯпјүгҖӮ гҒ“гҒ®дҪңе“ҒгӮ’иҰӢгҒҰгҖҢгғҠгғҲгғӘгӮҰгғ гҒ®еЎҠгӮ’жұ гҒ«жҠ•гҒ’иҫјгӮҖгҒЁгҒӢпјҢгӮ„гҒЈгҒҰгҒҝгҒҰгғјгҖҚгҒЁгҒӢжҖқгҒЈгҒҹгҒӘгӮүиІҙж–№гҒҜз§‘еӯҰиҖ…гҒ«еҗ‘гҒ„гҒҰгҒ„гӮӢпјҲгҒӢгӮӮгҒ—гӮҢгҒӘгҒ„пјүгҖӮ

гҒ“гҒ®дҪңе“ҒгҒҜжј«з”»гҒӘгҒ®гҒ§пјҢгҒҹгҒ йқўзҷҪгҒҠгҒӢгҒ—гҒҸиӘӯгӮҒгӮҢгҒ°гҒқгӮҢгҒ§гҒ„гҒ„гҒ®гҒ гӮҚгҒҶгҒ‘гҒ©пјҢеҖӢдәәзҡ„гҒ«гҒҜгҒӮгҒЁгҒҢгҒҚгҒ«еҜ„гҒӣгӮүгӮҢгҒҰгҒ„гӮӢж–Үз« гҒҢе°‘гҒ—ж•‘гҒ„гҒ«гҒӘгҒЈгҒҰгӮӢгҒӘпјҢгҒЁгҒҜжҖқгҒЈгҒҹгҖӮ

гҒқгҒҶгҒқгҒҶгҖӮгҒ“гҒ®дҪңе“ҒгҒҢйқўзҷҪгҒ„гҒЁжҖқгҒЈгҒҹдәәгҒ«гҒҜпјҢд»ҘдёӢгҒ«зҙ№д»ӢгҒҷгӮӢжң¬гӮӮгҒҠеӢ§гӮҒгҒҷгӮӢгҖӮ

Flickr Pro が値上がりしてた

Tags: #flickr #market #photography
 : 
History in GitHub Page

どうも，天下無敵の貧困者 Spiegel だよ。 派遣会社から送られてきた源泉徴収書を見て軽く絶望中です。

閑話休題 (それはさておき) 。

今朝 Flickr からメールが来てて，どうやら Flickr Pro の料金を値上げするらしい。 こんな感じ。

これは安易に Pro ユーザをお奨めできなくなったなぁ。

ただし現時点で Pro ユーザで2年分一括で払ってくれるなら今までと同じ値段にしてやると言われ，払い込みましたよ，さっき。

まぁ Flickr 側の苦しい台所事情も分かるし，私自身も確かに

と言ったし，払えない金額でもないので払うけどさ。 なんちうかこう，喫煙嗜癖者に対して煙草代をちまちま値上げして真綿で絞め殺す，みたいなやり口は尺然としない。

とにかく，2022年3月までは Pro ユーザらしいので，今後ともよろしく。

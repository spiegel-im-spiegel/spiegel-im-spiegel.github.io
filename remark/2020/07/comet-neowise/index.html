C/2020 F3 (NEOWISE) が見頃

Tags: #astronomy #commet
 :     
History in GitHub Page

2020年3月に NEOWISE によって発見された彗星 C/2020 F3 (NEOWISE) が明け方の東の空で見頃になっているらしい。 私は朝が弱いので，明け方の天文イベントはダメダメなんだよなぁ orz

ちうわけで記念に記事を残しておく。 以下は ISS から見た NEOWISE 彗星だそうだ。

この動画の3分過ぎあたりから NEOWISE 彗星が昇ってくる。

ちなみに NEOWISE は NASA の WISE (Wide-field Infrared Survey Explorer) 宇宙望遠鏡を使って地球近傍天体（Near-Earth Object; NEO）を捜索するためのミッションおよび宇宙機を指している。 ちゃんと仕事して成果を出してるわけやね。
